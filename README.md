# Registration Form (HTML5)

Ce projet consiste à créer un formulaire d'inscription avec validation HTML5.

## Objectif
Créer un formulaire accessible avec différents types de champs et validations.

## Champs du formulaire
- Nom et prénom
- Email
- Téléphone
- Date de naissance
- Mot de passe avec validation
- Sélection du pays
- Genre (radio buttons)
- Centres d'intérêt (checkboxes)
- Biographie
- Upload de photo de profil

## Fonctionnalités
- Validation HTML5
- Attributs required
- Pattern pour mot de passe et téléphone
- Organisation avec fieldset

## Technologies utilisées
- HTML5
- CSS
